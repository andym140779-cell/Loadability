# B777F Oversize Acceptance Planner

Open `index.html` in a browser to run the planner.

The calculator is based on the attached B777F workbook's visible rules for:

- side cargo door length/width/height envelope
- main deck 65.3 and 118.8 kg per inch zone limits
- 136 kg/sq ft area load limit
- base pallet usable footprint and tare weights
- shoring thickness, width, length, and 3% shoring weight allowance
- lock versus additional tie-down/strapping thresholds from the 777F limits tab

This is a screening tool for fast acceptability decisions. Final loading remains subject to operator, engineering, and airline load-control approval.
